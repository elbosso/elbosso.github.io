Ant erweitern mittels Javascript

vorhergehende Artikel in: Java
15.02.2015

Wie man Ant einfach erweitern kann, ohne wirklich Erweiterungen erstellen zu müssen...

Eine Aufgabe, die ich gern portabel und systemübergreifend in ANT realisiert hätte, stellte sich als schwieriger heraus, als ich gedacht hätte: Ich wollte eine während des Build-Prozesses generierte Datei nach ihrer Erstellung in einer passenden Anwendung zur Vorschau öffnen.

Leider existiert so ein Task nicht im Standard-Repertoire von ANT. Bevor ich anfing, einen eigenen zu schreiben, der die Funktionalität von java.awt.Desktop in ANT verfügbar machen würde, fand ich heraus, dass es sich doch lohnen könnte, JavaScript zu lernen:

Die Vorschau der mittels eigenen Content Management Systems erstellten Webseite realisierte ich wie folgt: 

    <script language="javascript">
        <![CDATA[
            var location = "file:///"+project.getProperty("basedir").toString().replaceAll("\\\\","/")+"/generated/index.html";
			location = java.net.URLEncoder.encode(location, "UTF-8");
			location = location.toString().replace("%3A",":");
			location = location.toString().replace("%2F","/");
            var uriLocation = java.net.URI.create(location);
            var desktop = java.awt.Desktop.getDesktop();
            desktop.browse(uriLocation);
        ]]>
    </script>
Für die Vorschau der erzeugten DocBook-Handbücher definierte ich mittels Javascript einen Task: 
	<scriptdef name="open" language="javascript">
		<attribute name="file" />
		<![CDATA[
			var location = "file://"+attributes.get("file").toString().replaceAll("\\\\","/");
			location = java.net.URLEncoder.encode(location, "UTF-8");
			location = location.toString().replace("%3A",":");
			location = location.toString().replace("%2F","/");
			var uriLocation = java.net.URI.create(location);
			var desktop = java.awt.Desktop.getDesktop();
			desktop.open(new java.io.File(attributes.get("file")));
		]]>
	</scriptdef>
den ich dann wie folgt benutzen kann: 
<open file="${prefix}/doc/output/handbook.pdf"/>
