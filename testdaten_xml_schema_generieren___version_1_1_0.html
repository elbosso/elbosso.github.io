Testdaten für ein XML Schema generieren - Version 1.1.0

vorhergehende Artikel in: Java Software-Test Komponenten
07.05.2023

Als ich neulich nach Möglichkeiten suchte, Rezepte elektronisch aufzubereiten und zu präsentieren stieß ich auch auf einige proprietäre und nicht mehr ganz druckfrische XML Schemata zur Beschreibung solcher Datenstrukturen.

Diese benutzte ich, um weitere Tests für meinen auf XML Schema beruhenden XML-Dokumentengerator zu verfassen. Es handelte sich dabei unter anderem um CookML und die Recipe Exchange Markup Language.

Nachdem ich diese beiden neuen Schemata in die Unit-Testsuite integriert hatte, nutzte ich die Gelegenheit, einige Lasttests durchzuführen und stieß dabei in einem der bereits in den Tests eingeflochtenen Schemata für biometrische Anwendungen auf einen Fehler, den ich umgehend an die entsprechende Mailing List meldete. Ich glaube zwar nicht, dass ich da jemals Feedback dazu erhalten werde (die letzte Aktualisierung dieses Schema fand 2014 statt) - aber es ist gut zu wissen, dass mein Generator tatsächlich auch in komplexen Schemata zur Fehlersuche und Validierung genutzt werden kann...
