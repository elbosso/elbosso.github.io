sQLshell Version 7.5 bringt Unterstützung für den Datentyp XML

vorhergehende Artikel in: Java Komponenten sQLshell Markup GUI
03.09.2022

Eine neue Version der sQLshell ist verfügbar!

Die neue Version der sQLshell bringt Unterstützung für den Datentyp XML für Datenbanken, die diesen Datentyp unterstützen.

Diese Unterstützung ist in allen Bereichen der sQLshell aktiv - speziell die Möglichkeiten zum Editieren und ansehen entsprechender Zeileninhalte berücksichtigen diesen Datentyp - es sind dafür spezielle GUI-Komponenten zur Inspektion und zur Änderung von XML-Daten verfügbar. Ein Beispiel dafür ist hier zu sehen:

XML-Editor mit geöffnetem Dialog zur Suche nach XPath-Ausdrücken

Der Editor, der als Komponente zur Manipulation von XML-Daten angeboten wird verfügt neben Syntax-Highlighting für XML-Dokumente über einige Spezialfunktionen - hervorzuheben ist hier die Suche nach XPath-Ausdrücken, wie sie auch in der oben stehenden Abbildung zu sehen ist.
