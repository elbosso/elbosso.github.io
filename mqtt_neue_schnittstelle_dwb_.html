MQTT als neue Schnittstelle in dWb+

Updates
  • 25. April 2015

Ich habe hier bereits in verschiedenen Artikeln über Möglichkeiten von Enterprise Integration in dWb+ berichtet - unter anderem dieser hier. Nunmehr ist eine weitere hinzugekommen: MQTT

Es existieren verschiedenste Client-Bibliotheken zur Interaktion von Komponenten mittels MQTT. Ich habe für einen schnellen Test zwei davon ausprobiert: Zwei Programme sollten sich mittels MQTT unterhalten - eines davon in Java geschrieben und eines in Python.

Zur Erleichterung der Programmierung habe ich den MQTT-Client benutzt. Die zum Senden respektive Empfangen verwendeten - diesem Beispiel hier stark nachempfundenen - Python-Scripts sind an diesem Artikel angehängt. Als Broker kam Apache ActiveMQ zum Einsatz.
