SVG -> Icon

vorhergehende Artikel in: GUI
30.04.2013

Von Hand Icons aus SVG-Graphiken zu erzeugen mag machbar sein, wenn man eine Graphik hat und diese in eine festgelegte Auflösung exportieren möchte. Hat man jedoch mehrere Graphiken und als Ziel verschiedene Auflösungsstufen, wünscht man sich schnell einen Automatismus.

Dieses Ziel kann man natürlich mit Software erreichen, die man zu Hauf im Web findet. Aber ich hatte noch eine weitere Randbedingung, die über das bloße Konvertieren von SVG nach Bitmap hinausging: Beim Rendern in verschiedenen Auflösungsstufen gehen unter Umständen Details verloren: Ist eine Graphik für eine Auflösung von 48x48 Pixeln vorgesehen und enthält sie Linien, die nur einen Pixel breit sind, werden diese Linien beim Erzeugen einer Bitmapgraphik verschwinden, die nur 32x32 Pixel groß ist.

Daher habe ich unter Zuhilfenahme von Batik einen Konverter geschrieben, der das Ausgangsdokument untersucht und zu dünne Linien entsprechend "verdickt", daß sie im Resultat mindestens 1 Pixel breit sind - dadurch gehen feine Details auch auf niedrigeren Auflösungsstufen nicht verloren!
