NFC-Tags als Trigger in Linux-Scripts

vorhergehende Artikel in: Linux
02.12.2016

Nachdem ich meine neue Liebe für NFC entdeckt habe und unter anderem das entsprechende PAM-Modul ausprobiert habe, wollte ich nun versuchen, NFC-Tags in Linux-Skripten als Trigger einzusetzen.

Ich wollte dazu nichts programmieren - ich dachte mir, dass es bereits Tools geben müsse, die mir die Arbeit abnehmen können. Und ich hatte recht! Folgendes Expect-Script gibt eine erste Idee davon, was möglich ist: 

#!/usr/bin/expect
set timeout -1
spawn pcsc_scan 
set card 0
set ATR ""
while {1 > 0} {
expect "Card state: Card inserted," {
puts stderr "\nKarte aufgelegt"
set card 1
}
expect -re "ATR:(.*)" {
if { $card >0} {
puts stderr $expect_out(1,string)
set ATR $expect_out(1,string)
}
}
expect "Card state: Card removed," {
puts stderr "\nKarte entfernt"
if { $card >0} {
if { {string trimleft $ATR} != ""} {
puts stderr $ATR
set card 0
set ATR {}
}
}
}
}
