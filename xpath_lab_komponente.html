XPath-Lab-Komponente

vorhergehende Artikel in: Java Komponenten GUI
10.12.2017

Ich habe als Fingerübung eine Komponente erstellt, mit der man XPath-Ausdrücke am lebenden Objekt studieren kann...

Das Originaldokument wird in der linken Hälfte der Komponente dargestellt. Die über den XPath-Ausdruck gefilterten Knoten werden in einer Baumansicht rechts daneben präsentiert. Diese Darstellung kann über einen regulären Ausdruck angepasst werden: alle capturing groups werden in das Template integriert und zur Darstellung der Knoten verwendet:
